# 🛡️ Rakshak AI

Rakshak AI is an AI-powered public safety and emergency assistance platform designed to enhance citizen safety through intelligent features, real-time alerts, and quick access to emergency resources.

## ✨ Features

- 🚨 Emergency assistance dashboard
- 🤖 AI-powered safety support
- 📍 Location-based emergency services
- 📞 Quick emergency contact access
- 📱 Responsive and user-friendly interface

## 🛠️ Tech Stack

- React
- TypeScript
- Vite
- Tailwind CSS
- AI Integration



## 📦 Installation

```bash
git clone https://github.com/killverted/rakshak-ai.git
cd rakshak-ai
npm install
npm run dev
```
